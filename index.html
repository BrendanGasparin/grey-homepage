W3Schools

W3Schools.com is an essential free resource for web developers from newbies to pros. The tutorials are excellent, especially for beginning coders, and the documentation sections on various different languages and technologies are necessary bookmarks for web developers of all levels. In fact, whether or not you are using the tutorials, you are sure to frequently encounter the W3Schools documentation just from googling your web development problems. If you're a beginner, start with the HTML tutorial.

2021-01-18

freeCodeCamp.org

freeCodeCamp.org is another free resource for learning web developers. I highly recommend it to anyone who wants to learn web design from scratch, as well as those wanting to refresh or update existing skills.

2021-01-09

The Odin Project

The Odin Project Foundations module is a great free resource.

It teaches:

  • Linux (including installation, dual-booting, and virtual machines).
  • Git and Github.
  • Web design (HTML, CSS, JavaScript).

The Odin Project is free, open source, and links to many other valuable resources such as freeCodeCamp. It is ideal for anyone who wants to learn about web design, as well as Linux, virtual machines, Git, and Github. Because the subject scope is wider, it is a little more advanced than the tutorials on the previously mentioned websites.
