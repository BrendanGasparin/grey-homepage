The Odin Project

Currently I am working through The Odin Project Foundations module.

It teaches:

  • Linux (including installation, dual-booting, and virtual machines).
  • Git and Github.
  • Web design (HTML, CSS, JavaScript).

The Odin Project is free, open source, and links to many other valuable resources such as freeCodeCamp. It is ideal for anyone who wants to learn about web design, as well as Linux, virtual machines, Git, and Github.
